Week 8: CNC Milling, Molding, and Casting

Task: Design something and make it using CNC. Create something by molding and casting.


Skills used: 3d Modelling, SRM-20, Silicone Casting

This week I wanted to create a silicone version of the same detail that I used as a print test in week 5. The detail is something that would typically be carved in wood. I was curious about how the same object would turn out with 3D printing versus casting.

I first used the CNC to create a mold. The first pass was not detailed enough to clearly capture the curves and indentations of the form, but the second pass turned out well.
